Car Manual RAG Chatbot

This project demonstrates a Retrieval Augmented Generation (RAG) chatbot built using LangChain and Chroma.

The system loads an HTML car manual, splits it into chunks, embeds the content, and retrieves relevant sections to answer user questions about car warning messages.

Requirements:
- Python 3.10+
- OpenAI API key set as environment variable OPENAI_API_KEY

Run:
python main.py
